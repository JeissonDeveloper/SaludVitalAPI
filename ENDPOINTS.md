# ENDPOINTS DE LA API

## Registro de Usuario
- **URL:** `http://localhost/SaludVitalProyecto/SaludVitalAPI/registro.php`
- **Método:** POST
- **Descripción:** Permite registrar un nuevo usuario en la base de datos.
- **Cuerpo de la solicitud (JSON):**
  ```json
  {
      "tipo_documento": "CC",
      "documento": "123456789",
      "password": "mi_contraseña_segura"
  }
